# Formats a multi-line string as it it were on one line

Formats a multi-line string as it it were on one line

## Usage

``` r
single_line_str(text)
```

## Source

[Mangiola et
al.,2023](https://www.biorxiv.org/content/10.1101/2023.06.08.542671v3)

## Arguments

- text:

  Any character vector

## Value

The same character vector, with newlines and subsequent whitespace
removed
