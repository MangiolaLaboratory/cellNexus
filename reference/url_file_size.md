# Gets the file size of a number of remote files

Gets the file size of a number of remote files

## Usage

``` r
url_file_size(urls)
```

## Source

[Mangiola et
al.,2023](https://www.biorxiv.org/content/10.1101/2023.06.08.542671v3)

## Arguments

- urls:

  A character vector containing URLs

## Value

The file size of each of the files pointed to by the provided URL, in
gigabytes, as double vector
